How To Use Sammati?

Published on 12-Dec-2021

A NEW ERA OF VOTING IS ON THE HORIZON !

Sammati proposes a new system of voting where everything is transparent and no question of tampering the voting machine as it doesn't exists in first place.

Sammati is the upcoming way of casting vote where all the votes will be stored in the Blockchain, which makes the system tamper proof and fully transparent.

This idea is a Proof of Concept. This project has scope for furthur modifications.

How to use?

1. Install MetaMask Browser Extension.

2. Make sure that the Ethereum Account which you've been assigned is imported into MetaMask.

3. Visit the Cast Vote Page and cast your vote
